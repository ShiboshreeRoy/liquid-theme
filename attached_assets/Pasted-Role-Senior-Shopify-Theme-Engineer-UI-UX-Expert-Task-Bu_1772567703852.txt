Role: Senior Shopify Theme Engineer & UI/UX Expert.

Task: Build a production-grade, Online Store 2.0 compliant Shopify theme inspired by the iOS 17 Pro Max interface. The theme must be "Real-World Ready," meaning it follows all 2026 Shopify performance, accessibility (WCAG 2.2), and merchant-customization standards.

1. Programming Stack:

Templating: Liquid (Shopify’s engine).

Styling: Tailwind CSS (configured for premium dark mode).

Interactivity: Vanilla JavaScript (ES6+) for high performance (no jQuery).

Data: JSON for templates and schemas.

2. Real-World Directory Structure:
Create the following mandatory folder structure:

/layout: theme.liquid (The master wrapper with iOS-style frosted glass header).

/templates: index.json, product.json, collection.json, cart.json (JSON templates are required for Online Store 2.0).

/sections: announcement-bar.liquid, bento-hero.liquid (iOS style), featured-collection.liquid, main-product.liquid, footer.liquid.

/snippets: product-card.liquid (with rounded-3xl and haptic hover effects), icon-pack.liquid (SF Symbols style).

/config: settings_schema.json (Define merchant-editable colors like 'Titanium Silver' and 'Deep Black').

/locales: en.default.json (Required for translations).

/assets: application.js, tailwind-output.css.

3. Professional GUI Features:

Dynamic Island Header: A sticky, centered navigation element that expands on scroll or click.

Bento Grid Layout: Homepage sections must use the Apple-style modular grid.

Typography: Use the San Francisco system font stack for that native iOS feel.

Performance: Implement Lazy-loading for images and defer for all scripts to ensure a 90+ Lighthouse score.

Accessibility: Ensure all interactive elements have focus states and meet WCAG 2.2 contrast ratios.

4. Functional Logic:

Implement an AJAX Cart Drawer that slides in from the right with a "Glassmorphism" background.

Ensure the product.json template supports App Blocks, so merchants can install plugins without breaking the code.

Output: Generate the complete file tree and the full code for layout/theme.liquid, templates/index.json, and sections/bento-hero.liquid.

Why this is "Real-World Ready":
Online Store 2.0 (JSON Templates): Shopify no longer recommends .liquid files for templates (like index.liquid). Using .json allows merchants to drag and drop sections in the Theme Editor.

Performance: By specifying Vanilla JS and Tailwind, you avoid "theme bloat," which is the #1 reason stores fail speed tests.

App Compatibility: The prompt includes support for App Blocks, which is mandatory if you want your theme to work with modern Shopify apps from the App Store.
